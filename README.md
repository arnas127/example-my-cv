# Ema Simkutė — Jekyll (GitHub Pages) profile site

A single-page, bilingual (EN/LT) profile website built for GitHub Pages.

## Quick start (local)

Prereqs: Ruby + Bundler.

```bash
bundle install
bundle exec jekyll serve
```

Open: http://localhost:4000


## Edit text (translations)

All visible text lives in `_data/i18n.yml`.

- English: `en:`
- Lithuanian: `lt:`
